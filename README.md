# Instruções de build

Para compilar binários localmente:

```
make build
```

Para construir via Docker:

```
make docker-build
```

# Execução dos programas

```
make docker-run-prog
```

Onde prog é o nome do progama que quer rodar (upload,findrec,seek1,seek2).
